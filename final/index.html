So you want to use Google Maps?

If you're here that means you want to be able to make calls to the Google Maps API to help the functionality of your existing website. It's important to note that you should have some basic JavaScript knowledge before attempting this. Click the button below if you need to learn JS. The reason I chose Google Maps is because of how useful it is. The truth is that Google has some pretty good documentation, but it's very dense. My goal of this is to only hit the essential parts so a novice can implement a map on their website.

Usage Limits

Usage limits are a common question when dealing with an API. We all want to know how much we can use this feature without getting billed. According to Google's page your usage will be free until you exceed 25,000 maps loads per 24 hours for 90 consecutive days. Below is a direct link to their entire usage area.
